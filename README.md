---

在第二题中运用了稀疏数组的思想

如果要将两个空间进行处理然后产生一个新的结构相同的空间，这时候如果想要不增加新的空间，我们通常会将原先的一个空间扩展变成处理后的数据空间容器。

*如果采取这样的方法，我们最好考虑在存储时将处理好的数据放到较为稀疏的一侧（一般就在扩展空间中），在题目中就是从尾部开始放入数据。这样我们就不用担心在处理的过程中反复更改数据空间的结构*
